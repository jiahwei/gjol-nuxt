# GjoL DB

古网公告数据统计网站前端项目，项目通过爬虫技术自动收集游戏公告内容，依据版本分类公告，并对公告段落进行无监督文本分类。

## 🌐 项目地址

访问网站：[gjoldb.info](https://gjoldb.info)

## 安装依赖

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

## 运行项目

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev

# bun
bun run dev
```
